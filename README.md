📌프로젝트 소개

"올해는 정말 역대급이네." 매년 여름과 겨울이면 습관처럼 내뱉는 말입니다. 하지만 정말 이상한 점이 있었습니다. 숨이 턱 막히는 폭염 경보가 뜬 주말에도, 코끝이 떨어질 것 같은 한파가 몰아치는 날에도 예술의 전당 앞은 여전히 사람들로 붐볐기 때문입니다.

**"날씨가 이토록 극단적인데, 사람들은 정말 집에서 OTT만 보고 있을까? 아니면 오히려 이 기후를 피해 '문화적 피난처'를 찾는 것일까?"**

이 프로젝트는 단순히 데이터를 돌려보는 것이 아니라, 기후라는 거대한 변수가 우리 사회의 '노는 방식'을 어떻게 재편하고 있는지 확인해보고 싶다는 호기심에서 시작되었습니다.

-----

🪟프로젝트 필요

1. **여가 패러다임의 변화 포착:** 기후 위기가 단순히 생존의 문제를 넘어, 개인의 문화 향유 방식과 '삶의 질'에 어떤 실질적인 제약을 주는지 수치로 증명할 필요가 있습니다.
2. **문화 산업의 기후 회복력(Resilience) 탐구:** 극한 기후 상황에서 오프라인 문화 시설이 '기상 피난처'로서 어떤 가치를 지니는지 분석하여, 향후 문화 공간의 운영 및 마케팅 전략 수립에 기여하고자 합니다.
3. **데이터 기반의 행동 예측:** 막연한 추측이 아닌, 실제 기온과 방문객 수의 상관관계를 통해 기후 변화에 민감하게 반응하는 문화 소비층의 행동 패턴을 정의합니다.
